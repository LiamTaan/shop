# Shop Workspace

这是当前商城项目的工作区根目录，汇总了后端服务、管理端和移动端代码。

## 目录结构

- `shop-server`：后端服务与数据库脚本
- `shop-admin`：管理后台前端
- `shop-mobile`：商城移动端前端

## 当前定位

这是一个以商城业务为核心的多端项目工作区，仓库首页说明以当前业务为准。

## 开发建议

1. 先启动后端服务并完成数据库初始化。
2. 再联调管理后台与移动端。
3. 上线前统一检查域名、支付、微信、文件存储和消息配置。

## 备注

主目录已经按当前项目命名统一为 `shop-server`、`shop-admin`、`shop-mobile`。
