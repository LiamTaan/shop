# Shop Workspace

这是当前商城项目的工作区根目录，汇总了后端服务、管理端和移动端代码。

## 目录结构

- `ruoyi-vue-pro`：后端服务与数据库脚本
- `yudao-ui-admin-vue3`：管理后台前端
- `yudao-ui-mall-uniapp`：商城移动端前端

## 当前定位

这是一个以商城业务为核心的多端项目工作区，仓库首页说明以当前业务为准，不再使用原始模板项目介绍。

## 开发建议

1. 先启动后端服务并完成数据库初始化。
2. 再联调管理后台与移动端。
3. 上线前统一检查域名、支付、微信、文件存储和消息配置。

## 备注

部分目录名沿用了历史脚手架名称，后续如需要可以再统一做目录重命名与路径修正。
