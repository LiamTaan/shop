# 商城移动端

这是商城项目的移动端前端，基于 uni-app 构建，面向小程序和 H5 场景。

## 技术栈

- uni-app
- Vue 3
- TypeScript

## 支持场景

- 微信小程序
- H5
- 其它 uni-app 兼容端按需扩展

## 功能范围

- 首页、分类、搜索
- 商品详情、购物车、下单结算
- 订单、售后、地址管理
- 会员中心、优惠券、积分、营销活动

## 启动说明

1. 安装依赖。
2. 在 HBuilderX 或项目约定工具中导入本目录。
3. 配置接口域名与运行环境。
4. 编译到目标平台进行联调。

## 对接说明

- 默认对接用户端接口前缀：`/app-api`
- 登录、支付、分享等能力需要按目标平台补齐配置
- 发布前请核对小程序合法域名、隐私协议和支付参数

## 目录提示

- `pages`：业务页面
- `components`：通用组件
- `api`：接口请求
- `stores`：状态管理
- `uni_modules`：依赖扩展模块

## 备注

当前目录名称沿用历史结构，业务说明以当前商城项目为准。
